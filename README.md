# Chép Lời — Bản tải về

Phần mềm **chép lời** (chuyển giọng nói → văn bản) chạy **hoàn toàn ngoại tuyến** trên máy bạn — không internet, không giới hạn, không lộ dữ liệu.
Quà tặng từ **Coach Thắng Huyền Đức · 10X Life OS**. Trang giới thiệu: <https://10x-lifeos.com/chep-loi>

## Phiên bản

**v0.1.8** (17/07/2026) — Gọn hơn: chỉ xuất file phụ đề `.srt` (bỏ `.txt` thừa) · app tự báo khi có phiên bản mới. Kèm: tự bỏ qua file đã có transcript · nhãn backend trung thực.

## Cài đặt
- **Windows:** giải nén `.zip` → mở `.exe`. Nếu hiện *"Windows protected your PC"* → bấm **More info → Run anyway**.
- **macOS:** mở `.dmg` → kéo app vào **Applications** → lần đầu **chuột phải** app → **Open → Open**.
- **Lần đầu cần internet** để tải "bộ não" AI (~1.6 GB) về máy một lần; sau đó chạy **hoàn toàn ngoại tuyến**.

> App miễn phí, chưa mua chứng nhận nhà phát triển nên có cảnh báo ở lần mở đầu — hoàn toàn an toàn. (Mỗi bản tải đã kèm file *HƯỚNG DẪN CÀI ĐẶT*.)
